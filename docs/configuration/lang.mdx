# lang

The editor is available in three languages: **English (en)**, **spanish (es)** and **catalan (ca)**.

In order to change the language you have to set the 2 letter ISO language code to the configuration:

<Playground>
  {() => {
    const config = {
      lang: 'es'
    };
    const [editorState, setEditorState] = React.useState(MUIEditorState.createEmpty(config))
    const onChange = (newState) => {
      setEditorState(newState)
    }
    return (<MUIEditor editorState={editorState} onChange={onChange} config={config} />)
  }}
</Playground>

You can edit the code above and try other languages.
